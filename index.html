What is Syncany?

Syncany is a cross-platform file synchronization application that syncs folders between different client using any kind of backend storage. It aims to be a secure application to store files in the cloud and share files with others.

Sync your Files

Backup your photo collection and share files with friends.

Encrypted

Don't worry about your privacy. Files are encrypted before uploading.

Any Storage

Use any kind of storage - (S)FTP, WebDAV, and many more!
