Syncany

An open-source cloud storage and filesharing application.
Securely synchronize your files to any kind of storage!

What is Syncany?

Syncany is a cross-platform sync tool that synchronizes folders between different client using any kind of backend storage. It aims to be a secure application to store files in the cloud and share files with others. It is different from other alternatives in two main ways:

  • Data encryption: Syncany encrypts the files locally, so that any online storage can be used even for sensitive data.
  • Arbitrary storage: Syncany uses a plug-in based storage system. It can be used with any type of remote storage.
The current focus of the project is stability of the core. We support three types of storage:
  • Local Folder: uses any local folder as storage. This could be any mounted device, network file systems (NFS), or any virtual file system based on FUSE.
  • FTP: uses an FTP folder as remote repository.
  • WebDAV: uses one folder in a WebDAV as remote storage.
