What is Syncany?

Syncany is a cross-platform file synchronization application that syncs folders between different client using any kind of backend storage. It aims to be a secure application to store files in the cloud and share files with others. It is different from other alternatives in two main ways:

  • Data encryption: Syncany encrypts the files locally, so that any online storage can be used even for sensitive data.
  • Arbitrary storage: Syncany uses a plug-in based storage system. It can be used with any type of remote storage.
